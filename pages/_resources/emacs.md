## Installation
### Mac
```bash
brew update
brew install emacs
brew services start emacs
```

### Linux

```bash
sudo add-apt-repository universe
sudo apt update
sudo apt install emacs
```


## Useful References
* [Emacs Cheatsheet](https://www.gnu.org/software/emacs/refcards/pdf/refcard.pdf)

## Useful Commands

### Start emacs
`emacs my_file.txt`

### Quit emacs
`Ctrl+Z`
